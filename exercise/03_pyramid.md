# Pyramids

**Implement the following Python program:**

Output a pyramid using `*` symbols.
The height of the pyramid is defined by a user input.

## Examples

**Input**
```console
3
```
**Output**
```
  *
 ***
*****
```

---

**Input**
```console
5
```
**Output**
```
    *
   ***
  *****
 *******
*********
```
